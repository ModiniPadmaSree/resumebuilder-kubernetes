# Resume Builder Deployment using Docker & Kubernetes (EKS)

This repository demonstrates deploying a full-stack Resume Builder application
using Docker and Kubernetes on AWS EKS.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Containerization: Docker
- Orchestration: Kubernetes (EKS)
- Cloud: AWS 
## Prerequisites
- AWS EKS cluster
- kubectl configured
- Docker images available in Docker Hub
## Deployment Steps
kubectl apply -f backend/ \
kubectl apply -f frontend/
